
# Krishi Mitra




## Deployment

To deploy this project run

#### To initialize virtual environment in windows
```bash
  ./setup.bat
```

#### To initialize virtual environment in linux
```bash
  ./setup.sh
```


#### To initialize backend server
```bash
python3 ./backend/app.py
```
